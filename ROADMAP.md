# Roadmap

## Phase 1

- repository bootstrap
- batch ingestion connectors
- warehouse schemas and marts
- Airflow orchestration
- sample dashboards and KPI definitions

## Phase 2

- Kafka consumers for booking, fleet, and pricing topics
- richer data quality monitoring
- forecast feature generation and baseline demand model

## Phase 3

- external demand enrichments such as weather or local events
- pricing elasticity analysis
- branch-level anomaly detection
- semantic metric layer integration with BI tooling

## Phase 4

- data contract enforcement
- automated lineage publishing
- advanced cost monitoring and workload optimization
