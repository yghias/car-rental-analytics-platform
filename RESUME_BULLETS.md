# Resume Bullets

- Designed and documented an end-to-end car rental analytics platform integrating reservation APIs, fleet systems, pricing services, maintenance feeds, and Kafka event streams into a cloud warehouse with raw, staging, core, and mart layers.
- Built production-style Python ingestion services, Airflow orchestration, dbt-style SQL models, and warehouse marts for fleet utilization, booking analytics, revenue reporting, pricing effectiveness, and operational KPI delivery.
- Implemented near-real-time event processing patterns for booking, fleet status, and pricing updates with schema validation, deduplication, dead-letter handling, and replay-aware observability.
- Modeled canonical data entities and dimensional warehouse structures for bookings, vehicles, customers, locations, maintenance events, and forecasting features with strong lineage, source-of-truth, and governance controls.
- Defined cloud infrastructure, CI/CD, security, monitoring, and runbook practices suitable for Senior Data Engineer, Cloud Engineer, Database Engineer, DBA, and analytics leadership interviews.
